Section 01

OPアンプの基礎知識

OPアンプ（演算増幅器、Operational Amplifier）は、2つの入力端子と1つの出力端子を持つ集積回路（IC）です。今回の実験では LM301A という8ピンのICを使用します。

ポイント

OPアンプ単体では使えません。必ず ±12Vの電源 を接続して動作させます。また、実際の回路では「負帰還（フィードバック）」をかけることで安定した増幅器として機能します。

Section 03

実験1：反転増幅回路

📐

入力電圧 Vs に対して、出力電圧 Vout が 符号が反転し、一定倍に増幅 されることを確認します。増幅率（ゲイン）は抵抗の比 Rf/Rs で決まります。

回路の仕組み（まずここを理解しよう）

OPアンプの「−入力（反転入力）」に信号を入れます。G点という接続点を経由して、出力から入力へ信号を戻す「負帰還」をかけることで、安定した増幅回路になります。

イマジナルショート（仮想短絡）

理想OPアンプが正常に動いているとき、G点の電圧は常に 0V になります（アースには繋がっていないのに電位がゼロ）。これを「イマジナルショート」と呼びます。この原理からキルヒホッフの法則を使うと、増幅率が抵抗比 Rf/Rs になることが導けます。

回路図

図2：反転増幅回路　Rs = 10kΩ、Rf = 100kΩ、ゲイン = −10

接続手順（初学者向け詳細）

  1. 1
    電源OFFを確認してから作業開始
  2. 2
    可変電源の +端子 → Rs（10kΩ）の一端へ接続
  3. 3
    可変電源の −端子 → GND（共通グランド）へ接続
  4. 4
    Rs のもう一端（X点）→ OPアンプの −入力（ピン2） へ接続
  5. 5
    Rf（100kΩ）を OPアンプの 出力（ピン6） と −入力（ピン2） の間に接続（これが負帰還！）
  6. 6
    OPアンプの +入力（ピン3） → GND へ接続
  7. 7
    基板上の可変抵抗（オフセット調整用）には 何も接続しない
なぜ負帰還をかけるの？

OPアンプの増幅度Aは実際には数万〜数十万倍あります。この状態では少しの入力電圧でも出力が±12Vに張り付いてしまいます。Rfで出力を入力側に戻すことで、見かけの増幅率を Rf/Rs = 100kΩ/10kΩ = 10倍 に安定させます。

測定方法

テスター3台を同時に使うと効率的
各点とGNDの間に接続する
測定点テスターのプローブ位置備考
Vs（X点電圧）X点 と GND の間可変電源の出力電圧
VG（G点電圧）G点 と GND の間イマジナルショートの確認用
Vout（OUT点電圧）OUT点 と GND の間増幅された出力電圧

可変電源を −1.5V から +1.5V まで 0.1V 刻み で変化させ、その都度 Vout と VG を記録します。

解析のポイント

確認すること

① Vs に対する Vout のグラフが、傾き = −Rf/Rs = −10 の直線になること
② G点電圧 VG がほぼ 0V で一定であること（イマジナルショートの実証）
③ Vs が大きくなると出力が ±12V 付近で飽和（頭打ち）することを確認

反転増幅の理論式
Vout / Vs = −Rf / Rs = −100kΩ / 10kΩ = −10
Section 04

実験2：非反転増幅回路

📈

信号を +（非反転）入力 に入れることで、出力が符号反転せず（同符号のまま）増幅されます。増幅率は 1 + Rf/Rs になります。

実験1との違い

実験1では可変電源を「−入力側」に入れました。実験2では 入力の場所を入れ替えます。Rs と GND を直接繋ぎ、可変電源は「+入力側」に移動します。

回路図

図3：非反転増幅回路　Vs を + 入力に入れる　ゲイン = 1 + Rf/Rs = 11

接続手順

  1. 1
    実験1の可変電源を外す。Rs（10kΩ）の X点側を GND に直接接続 する
  2. 2
    可変電源の +端子 → OPアンプの +入力（ピン3） へ
  3. 3
    可変電源の −端子 → GND へ
  4. 4
    Rf（100kΩ）は実験1と同じく出力→−入力の間に接続したまま

測定・解析のポイント

Vs（Y点）、Vout、VG を各々と GND 間でテスター測定。Vs を −1.5V〜+1.5V で 0.1V 刻みに変化させながら記録します。

非反転増幅の理論式
Vout / Vs = 1 + Rf / Rs = 1 + 100/10 = +11
反転増幅との比較ポイント

① 出力の 符号が正 になる（Vs が正ならば Vout も正）
② ゲインが 1 + Rf/Rs = 11倍（反転増幅の10倍より1大きい）
③ G点電圧は0Vではなく Y点（Vs）と等しい値 になる

Section 05

実験3：加算回路（アダー）

➕

2つの入力電圧 V1（可変）と V2（乾電池1.5V固定）を足し合わせ、符号反転して出力します。Vout = −(V1 + V2) になります。

加算回路の仕組み

2本の抵抗（R1, R2）をそれぞれ異なる電圧源に接続し、もう一端を共通のG点（OPアンプの−入力）につなぎます。G点の電位は0Vに保たれているので、各抵抗を流れる電流の合計が帰還抵抗 Rf を流れます。これにより2つの電圧が加算されます。

回路図

図4：加算回路　V1（可変）＋ V2（乾電池1.5V）を加算して符号反転

接続手順

  1. 1
    R1 = R2 = Rf = 100kΩ とする
  2. 2
    乾電池（V2 = 1.5V）の +側 → R2 → OPアンプの −入力（G点） へ
  3. 3
    乾電池の −側 → GND へ
  4. 4
    可変電源の +端子 → R1 → OPアンプの −入力（G点） へ（R2と同じG点！）
  5. 5
    可変電源の −端子 → GND へ
  6. 6
    Rf（100kΩ）を出力 → −入力（G点）に接続（負帰還）
  7. 7
    OPアンプの +入力（ピン3） → GND へ

測定・解析

V2 をテスターで実測。V1 を 0.5V 刻みで変化させながら Vout を記録。V1、V2、Vout の関係をグラフに描きます。

加算回路の理論式（R1 = R2 = Rf のとき）
Vout = −( Rf/R1 × V1 + Rf/R2 × V2 ) = −( V1 + V2 )
Section 06

実験4：理想的整流器（半波整流回路）

〜

ダイオードと OPアンプを組み合わせて、通常のダイオード整流器より優れた 「理想的な」整流器 を作ります。入力電圧の正側だけを取り出します。

なぜ「理想的」なの？

普通のダイオードには約 0.6V の順方向電圧降下 があります。つまり入力が 0.6V 以下では出力が出ません。OPアンプと組み合わせることで、この電圧降下を実質的になくし、ほぼ0Vから整流できる 理想的な動作を実現します。

回路図

図5：理想的整流器　出力は W点の電圧 Vw を測定する（Z点ではない！）
⚠ 注意：出力の測定点は Z点（ピン6）ではなく W点 です。ダイオードが2個乗った専用の小基板を使って赤枠部分を配線します。

接続手順

  1. 1
    可変電源の +端子 → Rs（100kΩ）→ OPアンプの −入力（G点） へ
  2. 2
    可変電源の −端子 → GND へ
  3. 3
    OPアンプの +入力（ピン3） → GND へ
  4. 4
    ダイオード2個搭載の 専用基板 を用いて出力−入力間を接続（教科書図14参照）
  5. 5
    Rs = Rf = 100kΩ とする

測定・解析

Vs と GND の間、Vw（W点） と GND の間にテスターを接続。Vs を −1.5V〜+1.5V で 0.1V 刻みに変化させ、Vw を記録します。

考察のポイント

① Vs > 0 のとき：D2 が導通し W点に電圧が出る（Vw ≈ −Vs）
② Vs < 0 のとき：D2 が逆バイアスとなり W点 ≈ 0V
③ 各点（G, Z, W）の電圧の正負と電流の向きを考察すること
④ 普通のダイオード整流と比べ、ほぼ 0V から整流できることを確認

Section 07

2週目：交流回路（半波・全波整流）

2週目の目標

交流入力（ファンクションジェネレーター）を使い、半波整流・全波整流回路の入出力波形をオシロスコープで観察して、回路特性を評価します。

半波整流回路

1週目の実験4（理想整流器）と同じ回路に交流を入力。正の半波のみが出力に現れる波形をオシロスコープで確認します。

全波整流回路

半波整流回路 ＋ 加算回路を組み合わせて全波整流を実現。交流の正側・負側どちらも同符号に変換されることを確認します。
